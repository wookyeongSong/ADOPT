# ADOPT: Additive Optimal Transport Regression


## General Information

This repository contains the implementation for the paper **"ADOPT: Additive Optimal Transport Regression",**  *AISTATS* (2026), which were run using R 4.3.1. It provides R scripts that reproduce simulation results.

## Details on reproducing results

* `simulations_density.R`: Contains simulation results for distributional-valued responses with Wasserstein metric; Produce Table 1 in Section 5.1.
* `simulations_spd.R`: Contains simulation results for SPD-valued responses with log-Cholesky metric; Produce Table 2 in Section 5.2.
